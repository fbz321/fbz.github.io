Python 易错题

列表操作易错点：

  1. 列表索引从0开始：在 Python 中，列表的索引是从0开始的。如果一个列表有n个元素，最后一个元素的索引是n-1。
  2. 负数索引：负数索引表示从列表末尾开始计数，如list[-1]表示列表中的最后一个元素。
  3. 切片操作：列表可以用切片来获取子列表，切片操作包括起始位置、结束位置和步长。但是要注意切片操作的边界以及步长的使用方式。
  4. 列表方法：列表有许多内置的方法，如.append()、.extend()、.remove() 等。初学者可能混淆这些方法的作用和使用方式。

字典操作易错点：

  1. 字典的键值对应关系：字典是由键值对构成的，要正确理解和使用键值对这一特性。
  2. 访问字典元素：通过键来访问字典中的元素，如果键不存在会引发 KeyError 错误。
  3. 字典方法：字典也有许多内置方法，如.keys()、.values()、.items()等，初学者需要逐步掌握这些方法的用法。
  4. 字典的遍历：初学者在遍历字典时可能会混淆如何遍历键、值或键值对。

元组的易错点：

  1. 不可变性： 元组是不可变的，一旦创建就不能被修改。
  2. 单个元素的元组： 定义只有一个元素的元组时，需要在元素后面加上逗号，如single_tuple = (1,)。
  3. 元组方法： 元组拥有少量方法，如.index() 和.count()，但没有修改元组的方法。

集合的易错点：

  1. 无序性： 集合是无序的，其元素没有索引。
  2. 不重复性： 集合中的元素是不重复的。
  3. 集合操作： 初学者有时会混淆集合的操作，例如交集、并集和差集的使用方式。
  4. 可变集合： Python 中有一个名为集合（set）的可变数据类型，初学者有时会混淆可变集合和不可变集合（frozenset）。

Python内置函数

以下是一些常用的内置函数：

  • abs(): 返回数值的绝对值.
  • all(): 如果可迭代参数中的所有元素都为真，则返回 True.
  • any(): 如果可迭代参数中的任何元素都为真，则返回 True.
  • ascii(): 返回一个表示对象的可打印版本的字符串.
  • bin(): 将整数转换为二进制字符串.
  • bool(): 将给定参数转换为布尔类型.
  • bytearray(): 返回一个新的字节数组.
  • bytes(): 返回一个新的 bytes 对象.
  • callable(): 检查对象是否可调用.
  • chr(), ord(): chr() 接受一个整数输入，并返回一个对应的 Unicode 字符，ord() 接受一个表示单个字符的字符串，并返回对应的 Unicode 码点.
  • classmethod(): 用于定义类方法.
  • compile(): 将字符串解析为 Python 语句.
  • complex(): 用于创建一个值为实部 + 虚部j的复数或转化为一个复数.
  • delattr(): 用于删除属性.
  • dict(): 用于创建一个新字典.
  • dir(): 用于查找对象内所有的属性和方法.
  • divmod(): 返回一个包含商和余数的元组.
  • enumerate(): 用于将一个可遍历的数据对象(如列表、元组或字符串)组合为一个索引序列，同时列出数据和数据下标.
  • eval(): 用于执行动态创建的 Python 代码.
  • exec(): 用于执行动态创建的 Python 代码.
  • filter(): 用于过滤序列，过滤掉不符合条件的元素，返回一个迭代器对象.
  • float(): 用于将整数和字符串转换为浮点数.
  • format(): 用于格式化字符串.
  • frozenset(): 返回一个冻结的集合，冻结后集合不能再添加或删除任何元素.
  • getattr(): 用于返回一个对象属性值.
  • globals(): 返回当前全局符号表的字典.
  • hasattr(): 用于判断对象是否包含对应的属性.
  • hash(): 用于获取取一个对象（字符串或者数值等）的哈希值.
  • help(): 用于查看函数或模块用途的详细说明.
  • hex(): 用于将10进制整数转换成16进制，以字符串形式表示.
  • id(): 用于获取对象的内存地址.
  • input(): 用于接受用户输入的内容.
  • int(): 用于将一个字符串或数字转换为整型.
  • isinstance(): 用于检查一个对象是否是一个已知的类型.
  • issubclass(): 用于检查一个类是否是另一个类的子类.
  • iter(): 返回一个可迭代对象.
  • len(): 用于返回对象的长度.
  • list(): 用于将元组或字符串转换为列表.
  • locals(): 返回当前局部符号表的字典.
  • map(): 用于根据提供的函数对指定序列做映射.
  • max(): 返回给定参数的最大值.
  • memoryview(): 返回给定参数的内存查看对象.
  • min(): 返回给定参数的最小值.
  • next(): 返回迭代器的下一个项目.
  • object(): 用于创建一个空对象.
  • oct(): 将整数转换为八进制字符串.
  • open(): 用于打开文件及相应的文件操作.
  • ord(): 返回表示单个字符的 Unicode 字符的整数.
  • pow(): 用于幂运算.
  • print(): 用于打印输出.
